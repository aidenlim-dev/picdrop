# PicDrop - 프로젝트 요약

## 🎯 프로젝트 한 줄 요약
**QR 코드로 이벤트 사진을 모으고 실시간으로 공유하는 웹 앱**

## 📦 전달 파일

### 핵심 파일
1. **README.md** - 프로젝트 설명서 및 사용 가이드
2. **COMPLETION_REPORT.md** - 개발 완료 보고서
3. **requirements.txt** - Python 패키지 목록
4. **manage.py** - Django 관리 스크립트

### 테스트 도구
- `create_test_event.py` - 테스트 이벤트 생성
- `create_test_photos.py` - 테스트 사진 업로드

## 🚀 빠른 시작 (5분)

```bash
# 1. 프로젝트 디렉토리로 이동
cd projects/picdrop

# 2. 가상환경 활성화
source venv/bin/activate

# 3. 서버 실행
python manage.py runserver

# 4. 브라우저에서 접속
# - 홈: http://127.0.0.1:8000/
# - 관리자: http://127.0.0.1:8000/admin/ (admin/admin123)
```
